Mini-Games Android Mania

June 2019
Mini-Games Android Mania is a series of multiplayer Mini-games built in Java using Libgdx for the graphics. The Computer/Laptop, acts as the centre of the action, allowing the users to pick a game to play, and see what is happening. The users, can either play the game using keyboard controls, or use their phones as a controller. To utilize your phone as a controller, you will need to install the supplemantry androidmania app that we developed using android studio for Android phones (Version 5.0+). The app contains unique controlls for each of the 6 bundled games. Android Mania game use a Server/Client system, where the computer, as well as the phones are all clients that transfer information between each other using a Digital Ocean Server (unfortunately now offline). The games we created include: A unique version of space invaders, Frogger, A rope race game, A memory puzzle block match, a word search game, and a bomb defusing game.

Ripple

April 2019
A project I developed along my friend at MasseyHacks. Ripple is a game that provides a story-driven narrative set in a post-apocalyptic world where water was once taken for granted and polluted. Our game aims to raise awareness of the value of water and portrays through an interactive game where users can experience daily life activities without clean water. The game was developed in Python, using Pygame for the graphics. All illustrations were hand drawn in Photoshop (By my friend Lydia Tan).

Pokemon Arena

February 2019
Pokemon Arena is a command line text based JAVA game. By etnering commands, the user is able to control and command his Pokemon, in his journey to become the pokemon master. Pick between 28 different Pokemon of different types, each with their own unique attacks and special effects. Seeing as the game is fully based on command line text, I decided that it would not be fun without having colored text! So I Colored all the console output in an effort to make the game look as good as it is enjoyable. Pokemon Arena used a lot of Object Oriented Design elements, and was a very fun project to create!

SeizeControl

September 2018
As a part of Hack The North, my team and I decided to build something that could benifit people around the world. SeizeControl is a Chrome extension that filters frames of YouTube videos that have a high probability of causing a negative reaction for individuals with epilepsy. When a new video is visited, its ID is sent to Firebase Cloud Firestore to be relayed to a beefy Google Compute server. The video is then downloaded and screened by a Python script and openCV. Once each portion is given a score, the results are sent back to Cloud Firestore for future reference and for the client to process. The YouTube video is then played while SeizeControl monitors the timestamp in the background. If a portion exceeds a user set threshold, the section is skipped Although the implementation of the screening algorithm was hugely inefficient (only designed for a single threaded CPU when we had an 8 core Xeon and an Nvidia Tesla GPU), the product in the end did work and was able to filter a Taylor Swift music video.
